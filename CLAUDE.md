# よりみちクエスト（yorimichi-quest）

## 企画概要

**ターゲット**: 大人（20〜40代）の気分転換。毎日が仕事と家事のループになっていて、「無意味なことをする余白」を失っている人。子どもと一緒に遊んでも成立する。

**コンセプト**:
「ゴミをひとつ拾って捨てる」「ダンボールを丸く切る」「すべすべの石をひとつ拾う」——
そんな**役に立たない小さなクエスト**が毎日提示されるデイリークエストアプリ。
習慣化アプリが「あなたを成長させます」と迫るのに対し、これは何も約束しない。
為にならないからこそ気楽に手が動き、やってみると なぜか少しだけ機嫌が良くなる。

**キャッチコピー**: 「人生に、意味のない用事を。」

**設計原則（アンチ生産性）**:
- 効能を謳わない。「リラックスできます」「集中力が上がります」等の文言をUIに一切出さない
- ストリーク（連続日数）なし。何日サボっても何も起きない、責めない
- 完了は自己申告でOK。証明を求めない
- クエストは30秒〜5分で終わり、お金がかからないものだけ

## コアループ

1. アプリを開くと**クエストカードが1枚だけ**、縦長のTinder風スタック（後ろに山が見える）で表示される（3枠案→1枚に変更、2026-07-09）
2. 気に入らなければ**リロール（引き直し）無制限**。ボタンでもカードの左右スワイプでも引ける
3. やったら「やった！」で完了。**次のカードが出てくる**（＝1日の消化数は無制限）
4. 完了時に**写真を1枚だけ任意で添付**できる → アルバムに溜まる

### リロール無制限の採用について（決定事項）

無制限を採用する。理由: リロールに制限を付けると「もったいないから引けない」という圧が生まれ、ゆるさというコンセプトの核と矛盾する。また「石を拾う」は都心では無理、「ダンボールを切る」は手元にないと無理、といった実行可能性のばらつきを吸収する安全弁がリロールなので、ここをケチると詰みが発生する。

ただし「プール全体を眺めてタスクを消費し尽くす」暴発を防ぐため、以下でガードする:

- **1枚ずつ、スライドアニメーション付きで引く**（一覧性を与えない。カードを引くガチャ感が自制を促す）
- **その日リロールで流したクエストは、同日中は再登場しない**
- **デッキは日替わりシャッフル**（日付シードで決定的に生成。「見尽くした感」の起点を作らない）
- クエスト一覧画面は作らない。プールの中身は遊びながらしか見えない

## 継続の仕掛け（責めないコレクション）

- **おもいでアルバム**: 任意添付した写真がクエスト文と一緒に並ぶ。「丸く切ったダンボール」「すべすべの石」の写真が溜まっていく、意図せず面白いアルバムになる
- **むいみ週報**: 週に1回、無意味な統計を表示。「今週あなたは石を2つ拾いました」「猫の絵: 通算3枚」。真面目なグラフは出さない
- **称号**: 同カテゴリの累積で解锁（「すべすべ鑑定士」「ダンボール加工技師」など）。通知や達成圧はかけず、アルバムの隅にひっそり増える
- 進捗はローカル保存（localStorage）

## タスク文体ガイド（★ここがプロダクトの本体）

タスクの温度感がすべて。量産時に自己啓発臭・雑用臭に退化させないためのルール:

1. **数を指定する** — 「ひとつ」「1枚」「10歩」。数が入ると急にクエストらしくなる
2. **質感・条件をひとつ足す** — 「すべすべの」「まっすぐな」「利き手じゃないほうの」。この一語が詩になる
3. **効能を書かない** — 「〜してリラックス」は禁止。やる理由を説明しない
4. **言い切りの動詞で終える** — 「〜しよう」「〜してみませんか」ではなく「拾う」「切る」「描く」
5. **微妙に為になるものを2〜3割混ぜる** — ゴミ拾い・レシート整理など。ただし絶対に効能面を主張しない

- 良い例: 「アルミホイルで、できるだけまんまるの球を作る」
- 悪い例: 「深呼吸してリラックスしよう」（効能臭）／「部屋を掃除する」（大きい・義務・雑用）

## タスクタグ

| タグ | 値 |
|---|---|
| ばしょ | いえ / そと / どこでも |
| どうぐ | なし / 紙とペン / はさみ類 / スマホ |

**UIにはタグもフィルタも一切出さない**（2026-07-09、ユーザー判断で「極力シンプルに」へ方針決定。おうちモードと「ばしょ」表示は実装後に廃止）。タグはデータとして保持し、将来のバランス調整用。カテゴリチップも雑多カテゴリ「やってみる」は非表示。

## 初期タスクプール（36本・トーンの基準例）

### ひろう（そと）
1. すべすべの石をひとつ拾う
2. きれいな落ち葉を1枚拾って、本にはさむ
3. 落ちているゴミをひとつ拾って、捨てる
4. できるだけまっすぐな枝を探す

### つくる（いえ）
5. ダンボールを丸く切る
6. 紙飛行機を折って、部屋の端から端まで飛ばす
7. アルミホイルで、できるだけまんまるの球を作る
8. 輪ゴム3本で何かを発明する

### すてる・ととのえる（いえ）
9. いらないものをひとつ捨てる
10. 財布からレシートを全部出す
11. スマホの写真を、いらないもの1枚だけ消す
12. 机の上のものをひとつ、家の中の正しい場所に帰す
13. 手持ちのペンが全部書けるか試して、書けないやつを捨てる

### かく（どこでも）
14. 猫の絵を描く
15. 利き手じゃないほうの手で自分の名前を書く
16. 目をつぶって象の絵を描く
17. いまの気持ちを五七五で詠む

### みる・きく（どこでも）
18. 空を1分間見上げる
19. いちばん遠くから聞こえる音をつきとめる
20. 雲をひとつ選んで、名前をつける
21. 自分の手のひらのしわを30秒観察する
22. 今日の空の色に、いちばん近い色の名前をつける
23. 窓から見えるものをひとつ、声に出して実況する

### そと
24. 知らない道を1本だけ曲がってみる
25. 硬貨を投げて、表なら右、裏なら左に10歩歩く
26. 今日の空を写真に1枚撮る

### いえ
27. 飲み物をいつもと違うコップで飲む
28. 目をつぶって一口食べて、味を言葉にする
29. 家の中をいつもと逆回りで一周する
30. 家の中で、座ったことのない場所に座ってみる
31. 家にあるいちばん小さいものを探す
32. 本を1冊選んで、適当に開いたページだけ読む
33. 5年以上使っているものをひとつ見つけて、ねぎらう
34. ポケット（またはかばんの外ポケット）の中身を全部出して、並べる
35. その場で10秒だけ、全力で変なダンスをする
36. 靴下を左右わざと違うものにして過ごす

## 技術方針

- 単一HTML + CSS + Vanilla JS（既存プロジェクトと同様、ブラウザで開くだけで動く構成）
- クエストプールはJS内の静的配列（id, 本文, カテゴリ, ばしょ, どうぐ）
- 日替わりデッキは日付文字列をシードにした決定的シャッフル
- 写真はローカル保存（縮小してlocalStorage、容量が厳しければIndexedDB）
- 外部依存なし、オフライン動作、スマホ縦画面ファースト
